MNIST-1-数据集介绍

MNIST 数据集来自美国国家标准与技术研究所, National Institute of Standards and Technology (NIST). 训练集 (training set) 由来自 250 个不同人手写的数字构成, 其中 50% 是高中学生, 50% 来自人口普查局 (the Census Bureau) 的工作人员. 测试集(test set) 也是同样比例的手写数字数据.

包含了四个文件：

  • Training set images: train-images-idx3-ubyte.gz (9.9 MB, 解压后 47 MB, 包含 60,000 个样本)
  • Training set labels: train-labels-idx1-ubyte.gz (29 KB, 解压后 60 KB, 包含 60,000 个标签)
  • Test set images: t10k-images-idx3-ubyte.gz (1.6 MB, 解压后 7.8 MB, 包含 10,000 个样本)
  • Test set labels: t10k-labels-idx1-ubyte.gz (5KB, 解压后 10 KB, 包含 10,000 个标签)

数据格式

Training 集合中有 60000 张图片, 每一图片分辨都是 28*28 , 处理后的每一张图片是一个长度为 784 的一维数组.

labels 为对应图片的答案.

Test 为测试数据集, 虽然文件中并没有验证数据集, 通常会将training 分一部分为 Validating data.

图片像素取范围为 [0, 1], 0 为白色背景, 1 为黑色前景.

数据下载

  • MNIST 数据集可在 http://yann.lecun.com/exdb/mnist/ 中直接下载.

  • TensorFlow 提供了一个类处理 MNIST 数据， 可以自动下载并转化 MNIST 数据的格式.

    ​ #载入MNIST数据, 如果指定目录下没有数据, TensorFlow会自动从上述网址下载数据到指定文件夹.

    ​ from tensorflow.examples.tutorials.mnist import input_data

    ​ mnist = input_data.read_data_sets("dir", one_hot=True)
